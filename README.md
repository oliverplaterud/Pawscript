# Pawscript

En tavla bara för ditt djur – personliga husdjurstavlor med AI-genererade citat.

## Om projektet

Pawscript är en webbtjänst där husdjursägare kan:

1. Ladda upp en bild på sitt husdjur
2. Berätta lite om djurets personlighet
3. Låta AI generera ett unikt citat på svenska
4. Få tavlan tryckt och levererad hem

Varje tavla är unik – citaterna genereras specifikt för varje djur.

## Teknologi

- **Frontend:** HTML, CSS, JavaScript (vanilla)
- **AI:** Claude API (Anthropic) för citat-generering
- **Betalning:** Swish
- **Design:** Responisiv webbdesign med mobil-stöd

## Funktioner

- 📷 Bilduppladdning och förhandsvisning
- 🤖 AI-genererade personliga citat
- 🎨 Snygg tavla-preview före beställning
- 💳 Swish-betalning
- 📱 Mobilanpassad design

3. Följ instruktionerna för att skapa din tavla

## Notering om API

AI-funktionaliteten kräver en giltig Anthropic API-nyckel. Se till att sätta upp rätt autentisering innan du använder funktionen för citat-generering.
